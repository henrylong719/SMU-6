Overview

This page will provide a list showing the employee information. The list will be filtered by the condition above.

Navigation

Click Search Staff Information button on the top navigation bar to go to this section.

Filter The Condition

Name, location, department, and report to are the conditions which user can decide. When user change the searching condition. The result will be shown immediately.

The system would ignore the case of the letters, which means user do not need to fix the case of the first letter of the name.

The system will list all of the departments in the spreadsheet. User only can choose one of them. Once the condition changes, the list will refresh.

The system will list all of the locations in the spreadsheet. User only can choose one of them. Once the condition changes, the list will refresh.

Check the chart of the searching result

A chart of organization structure can be shown after condition setting. User can click chart button to see the chart.

The condition will be shown in the top. And organization structure will be shown in a pop-up window.

The initial chart will contain all the employee in the window, so that the words could be very small. Users can click “zoom in” and “zoom out” button to fix the size of the words.

If the chart cannot be completely shown in the window, there would be a scroll bar following the window. User can drag it to fix the content in the window (touchpad is supported).

If the mouse move over a block of an employee. A detail information will be shown as tips information.

Users can click the “X” button on the top-right to close this window.
